# KB-B · 构图与分层库(Composition & Layering Library)

> 决定画面的 **骨架** 和元素的 **空间位置**。解决「**好卖**」的问题。
> 调用逻辑:按 **直播类型(采销 / 达人 / 品牌)** + **素材组合** 匹配。

## 命中规则

按用户提供的素材判定:

```
有商品 + 无主播 + 无文字 → L-01 商品型
无商品 + 有主播 + 无文字 → L-02 主播型
无商品 + 无主播 + 有文字 → L-03 文字型(预热场景)
有商品 + 无主播 + 有文字 → L-04 商品+文字(采销主流)
无商品 + 有主播 + 有文字 → L-05 主播+文字(达人主流)
有商品 + 有主播 + 有文字 → L-06 全能型(品牌带货)
```

## 构图表

| ID | 直播类型 | 构图策略 | 分层架构(Z 轴) | 视觉重心 |
|---|---|---|---|---|
| **L-01** | 商品型<br/>(Product Only) | 微距舞台式<br/>(Macro Stage) | L1: 纯净微缩场景 / 底座<br/>L2: 商品(Z 轴核心)<br/>L3: 侧边极简信息栏 | **100% 商品**<br/>强调细节纹理 |
| **L-02** | 主播型<br/>(Host Only) | 人像特写式<br/>(Portrait Bokeh) | L1: 大光圈虚化背景<br/>L2: 主播(眼神光 / 中焦)<br/>L3: 氛围装饰(不挡脸) | **80% 人物**<br/>强调眼神交流 |
| **L-03** | 文字型<br/>(Typography) | 满版放射式<br/>(Full Bleed) | L1: 放射线 / 纯色高亮底<br/>L2: 无<br/>L3: 巨大 3D 文字撑满画面 | **100% 文字**<br/>强调紧迫感 |
| **L-04** | 商品 + 文字<br/>(Prod + Text) | 基座 / 分割式<br/>(Pedestal/Split) | L1: 场景背景<br/>L2: 商品位于基座 / 左侧<br/>L3: 文字作为基座 / 右侧排版 | **60% 商品 / 40% 文字**<br/>理性价值锚定 |
| **L-05** | 主播 + 文字<br/>(Host + Text) | 杂志穿插式<br/>(Interwoven) | L1: 纯色 / 质感墙面<br/>L2: 巨大文字(位于人物后)<br/>L3: 主播(遮挡部分文字) | **50% 人物 / 50% 概念**<br/>强调权威感 |
| **L-06** | 全能型<br/>(Host+Prod+Text) | 倒三角 / 手持式<br/>(Hand-held) | L1: 广角纵深背景<br/>L2: 主播手持商品向前递送<br/>L3: 顶部 / 底部大标题 | **人货合一**<br/>强调带货现场感 |

## 直播类型 → 推荐构图

| 直播类型 | 主选 | 备选 |
|---------|-----|------|
| 采销直播(强调卖货) | L-01 / L-04 | L-06(品牌也带货)|
| 达人直播(强调主播)| L-02 / L-05 | L-06 |
| 品牌直播(全方位) | L-06 | L-04 / L-05 |
| 预热直播(无人无货)| L-03 | - |

## 视觉重心规则(用户 0.5 秒锁定核心)

| 构图 | 视觉重心位置 | 安全区 |
|------|-----------|--------|
| L-01 | 画面正中 | 上下各留 8% 给信息层 |
| L-02 | 上 1/3 处眼神光 | 下 1/3 留给信息层 |
| L-03 | 画面正中 | 4 边各留 4% 防溢出 |
| L-04 | 左 60% / 右 40% 分割线 | 中线作平衡轴 |
| L-05 | 主播头部位置 | 字 + 人 5:5 视觉重 |
| L-06 | 黄金分割点(右下 1/3) | 主播手 → 商品 → 顶字 视觉流 |

## 边界

| 场景 | 处理 |
|------|------|
| 商品很多(N > 1) | 走 L-04 但用网格排列;N > 3 走 L-03 文字型 + 商品矩阵 |
| 主播 + 多个商品 | 走 L-06,商品在主播手中,数量 ≤ 2 |
| 主播是真人但要遮脸 | 走 L-05 但 mask 头部,通过眼神光暗示 |
| 商品是虚拟(如服务券) | 走 L-03 文字型 + 装饰 icon |
