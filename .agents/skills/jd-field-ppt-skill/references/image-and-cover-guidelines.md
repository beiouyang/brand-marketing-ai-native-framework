# 配图与封面

## 图片类型

- 真实业务截图：APP 页面、组件状态、商家后台、内容流、交易路径。
- 信息图：流程、系统关系、分层策略、指标结构。
- 数据图：柱状、折线、漏斗、象限，不做 3D 图。
- 场景图：用户购物、商家经营、履约、内容消费。要像真实业务现场，不像广告大片。

## 图片比例

- 封面主视觉：16:10 或 21:9。
- 页面截图槽：16:10。
- 右侧配图：4:3 或 1:1。
- 横向流程图：21:9。

## 生成提示词原则

- 中文 deck 的图中标签使用中文。
- 图片不自带标题、页码、角标、边框和长段说明。
- 截图保真优先，用户给了截图时不要默认重绘界面。
- 若需要生成封面，主标题优先，关键词不超过 18 个中文字符。

## 封面规格

- 演示封面：16:9，标题 + 副标题 + 业务对象视觉。
- 公众号头图：21:9，标题占左侧或中间，右侧保留视觉锚点。
- 1:1 分享卡：标题居上或居中，保留一个关键数字或业务对象。
