08 · 结构化表达

结构化表达：先分类，再矩阵化，再压缩信息

适合把一堆散点信息变成“层级、维度、规则、结论”。这不是某种图，而是一种组织信息的方法。

推荐路径：E + Deck 路径 F

一页压缩时，先建立信息架构。

对象

用户、商品、场域、触点。

动作

生成、排序、编排、投放。

约束

去重、频控、风险、体验。

结果

点击、转化、反馈、迭代。

结构矩阵
高确定性

已有明确意图，直接承接动作。

高探索性

用推荐词引导继续表达。

高风险

走过滤、解释、兜底。

高价值

进入重点触点投放。
