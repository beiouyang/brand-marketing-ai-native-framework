# 文字 · 变体 Variants

> design.md → [主文档](./design.md) · 同 bundle: [spec](./spec.md) · [behaviors](./behaviors.md)

## 变体维度概览

| 维度 | 取值 | 用途 |
|---|---|---|
| 实现方式 | 前端动态渲染 / 视觉切图配置 | 区分可变内容与视觉强化内容。 |
| 内容类型 | 阅读型内容 / 常规信息 / 视觉强化文案 / 数字表达 | 区分阅读效率、信息层级和营销表达强度。 |
| 字体类型 | 系统字体 / 图形化字体 / 风格字体 | 区分稳定可读与品牌氛围表达。 |
| 场景风格 | 品牌 / 通用 / 时尚 / 活力 / 传统 / 科技 | 根据品牌露出、品类、情绪和活动主题选择字体风格。 |

## 各维度详细规范

### 实现方式

| 类型 | 实现方式 | 适用场景 | 使用边界 |
|---|---|---|---|
| 前端动态渲染 | 系统字体 + 基础文字规范 | 商品信息、规则说明、可变权益、倒计时、用户个性化内容 | 不承载复杂图形装饰，不依赖切图传达核心可变信息。 |
| 视觉切图配置 | 图形化字体 / SVG / 图片资源 | 主视觉标题、广告、营销装饰文本、品牌氛围强化 | 不用于动态文本及阅读型内容。 |

### 内容类型

| 内容类型 | 推荐策略 |
|---|---|
| 阅读型内容 | 使用基础阅读型字号与系统字体，保证扫读效率。 |
| 常规信息 | 通过字号、字重、颜色区分标题、正文、辅助说明。 |
| 视觉强化文案 | 可使用更高强度的字号、字重、色彩或图形化字体。 |
| 数字表达 | 用于优惠、折扣、补贴、价格、倒计时等，仍归入基础阅读型体系。 |

### 字体风格

风格字体用于不同营销场景下的字体风格表达。风格化字体主要用于标题、主题文案及视觉强化内容，不用于阅读型文本内容。

| 风格 | 视觉特征 | 适用场景 | 推荐字体 / 实现 |
|---|---|---|---|
| 品牌字体 | 官方、可信、统一，体现平台品牌识别与品牌气质 | 京东官方品牌露出、品牌活动、平台级营销场景及需要强化官方认知的内容 | 品牌字体 / 京东朗正 2.0 / `jingdonglangzhengti2:Bold` |
| 通用风格 | 严谨、直接、高识别 | 全品类通用的 S 级大促（双11、618）的核心利益点、日常通栏、百亿补贴等强转化且需要极速阅读的场景 | 方正兰亭黑 简体、造字工坊方黑 / `FZLanTingHeiS:SemiBold`、`FZLanTingHeiS:Bold`、`造字工房方黑体:Regular` |
| 时尚风格 | 精致、轻盈、高级 | 美妆护肤、服饰箱包、奢侈品、会员日及女性向营销场景 | 造字工房逸锋体、造字工房书见体 / `造字工房逸锋体:Regular`、`造字工房书见体:Regular` |
| 活力风格 | 年轻、亲和、趣味 | 零食生鲜、母婴亲子、开学季、出游季、宠物节等年轻化营销场景 | 造字工房薄荷海盐体、造字工房简圆、得意黑 / `造字工房薄荷海盐体:Regular`、`造字工房简圆体:Regular`、`Smiley-Sans:Oblique` |
| 传统风格 | 文化感、节庆感、仪式感 | 春节、中秋节、端午节、国潮主题及传统文化营销场景 | 方正黑隶简体、汉仪永字蓬莱、汉仪杰龙桃花源 / `FZHeiLiS:Bold`、`HYYongZiPengLai:W`、`HYJieLongTaoHuaYuan:W` |
| 科技风格 | 理性、未来感、秩序感 | 3C 数码、家电、智能设备及科技主题营销场景 | 方正摩登体、造字工房启黑体 / `FZMoDengTiS:Bold`、`造字工房启黑体:Regular` |

### 信息层级组合

| 组合 | 说明 |
|---|---|
| 基础规范继承 + 营销扩展 | 先继承 JD APP 16.0 基础 Typography，再补充营销场景特有表达。 |
| 系统字体 + 图形化字体 | 系统字体负责动态渲染和阅读效率；图形化字体负责主视觉和品牌氛围。 |
| 数字表达 + 权益表达 | 数字字号扩展服务优惠、补贴、折扣、倒计时等营销信息。 |
| 字体风格 + 品类场景 | 不同风格字体匹配不同营销品类、情绪和活动目标，并通过案例切图稳定展示视觉效果。 |

## 章节原文（结构化摘录）

- 文字类型拓展包含前端动态渲染与视觉切图配置。
- 图形化字体通过视觉切图或图形资源配置实现。
- 图形化字体适用于主视觉标题、视觉强化文案、广告、营销装饰文本。
- 图形化字体不用于动态文本及阅读型内容。
- 设计稿包含品牌字体、通用风格、时尚风格、活力风格、传统风格、科技风格六组风格案例。
- 科技风格字体具有理性、未来感、秩序感，适用于 3C 数码、家电、智能设备及科技主题营销场景。
