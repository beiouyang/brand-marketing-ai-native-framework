文档版本：v2.0.1 · 生成日期：2026-05-29 · 工具：京东商详腰带 AI 改造工具

一、总览

工具定位：HEX / 场景 / 取色 → OKLCH → 腰带配色 + 内容编排 → 多端预览 → AI 规则审核 → 人工复核 → 上线

① 品牌色 → ② 价促信息 → ③ 氛围 icon → ④ 右侧 IP+利益点 → 腰带预览 → 商详预览 / JSON 导出 → 进入 AI 审核 → AI 自动审核 → 人工审核 → 发布上线（效果数据在数据看板沉淀）

二、腰带生产阶段（运营配置）

Step 1 · 品牌色来源（三选一）

模式功能诉求
颜色输入输入 HEX（#RRGGBB / RRGGBB / #RGB），点击「应用并映射」
场景推荐9 大业务场景（促销/外卖/超市/酒旅等），按 OKLCH 安全区间（H/L/C）选色
OKLCH 微调选中场景后可滑动调节 L、C，输出 HEX
图片取色上传图片 → OKLab 聚类 → 过滤 C/L → 点选候选色作为品牌色
自定义腰带背景上传背景图（建议 ≥750px 宽）；内容色可选「自动生成」或「自定义 HEX」

配色输出（scheme）

  • beltBg：腰带背景色
  • textPrimary / textSecondary：主/副文字色（OKLCH 自动推导）
  • logoColor：品牌 IP 着色
  • beltBgOklch：背景 OKLCH 值（便于审核）

前置条件：完成配色映射后，预览、导出、审核按钮才可用。

Step 2 · 价促信息（腰带左侧）

字段规则展示位置
价格必填，到手价两行/三行左侧主价区
划线价格必填副行（删除线）
优惠信息 1必填，≤9 字两行第 2 行 / 三行底栏
优惠信息 2必填，≤9 字与优惠 1 用竖线分隔
已售必现，≤12 字副行

功能诉求：左侧文案与预览实时同步；纯文本 + 竖线分隔（无闪电 icon、无箭头）；三行底栏为「已享受: 优惠1 | 优惠2」，右侧保留 8×8 跳转箭头。

Step 3 · 氛围 icon

能力说明
AI 生成固定提示词模板 + 可编辑变量（品牌元素/图形风格/主色/辅助色）
参考图支持多图上传
图库上传透明 PNG/WebP/SVG；内置「闪电氛围」示例
位置调节水平/垂直/缩放/透明度滑块
图层关系在腰带底色之上、价格与右侧文案之下

设计约束：偏右、大尺寸、低透明度、弱装饰，不遮挡文字。

Step 4 · 腰带右侧氛围区

档期与组合规范

档期IP 上限说明
大促期≤3可 3 logo 组合
日常≤2无三 logo 组合

右侧组合规范：

  • 01大促 IP / 增强心智
  • 02单 IP · 交叉 × · IP+氛围
  • 03Slogan / 倒计时 / 领券（利益点，仅 1 个）

整体约束：右侧 ≤3 行 · 利益点 1 个 · 增强心智 ≤1

品牌 IP 组合规则

模式结构尺寸
单 logo1 行高 14px，宽 ≤90px
双 logo上下 2 行每行 11px，间距 4px
三 logo上：大促；下：IP × IP11px；× 固定 11px（不缩小）
PC 端最多 2 个 logo 横向组合高 18px，442×52

利益点（单选）

类型优先级字数限制
倒计时P1 · 限时功能≤16 字

营销口号P3 · Slogan≤7 字

三、审核阶段（5 步流程）

审核引擎：v0.21（纯前端演示，无真实后端）

① 运营填写基础信息 → ② AI 自动审核 → ③ 人工审核 → ④ 发布上线
（AI 不通过 → 返回运营修改，不进入后续步骤）

① 运营填写基础信息

字段必填说明
项目名称✓非空、格式合规
业务线✓全品类/3C/家电/超市/时尚/健康
活动类型✓大促/S级/A级/B级/品类日等
上线端✓移动端 2 行 / 3 行 / PC（多选）
上线日期✓合法且不早于今天
提审人 ERP✓非空、格式合规

② AI 自动审核

类别检查项级别
配色背景色/主文字色缺失；对比度 WCAG AA ≥4.5:1；倒计时对比度P1/P2
LOGO/IP未选 IP / IP 超 3 个 / 组合拥挤P1/P2
右侧氛围区未选利益点 / Slogan 超 7 字 / 领券超 9 字 / 3IP+复杂利益点同屏P1/P2
氛围 icon非透明 PNG / 尺寸过小 / S+ 活动需复核P1/P2
排版文案截断 / 组合规则警告P2

审核结果分支

结果条件下一步
AI 不通过存在 P1返回运营修改
AI 通过但有风险有 P2 或需设计师复核发送设计师复审核
AI 通过无 P1/P2仍进入设计师复审核（演示策略）

③ 人工审核 → ④ 发布上线

  • 人工审核：展示审核员信息，发起审批流程；AI 不通过时可返回修改
  • 发布上线：演示流程完成后，效果数据在「数据看板」沉淀（含 Top10 腰带排名）
  • 支持「重新审核」「复制审核结果」
