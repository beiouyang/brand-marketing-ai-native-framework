# Campaign System Graphic Variants

## Variant Dimensions

| Dimension | Values | Purpose |
|---|---|---|
| Graphic role | Identification, information support, atmosphere | Defines why the graphic appears. |
| Graphic type | Platform shape, semantic, scene, theme, category | Defines the asset family and maintenance boundary. |
| State | Unselected, selected | Applies to marketing navigation / entrance icons. |
| Style expression | Flat, metal, glass, festive, category illustration | Allows visual skin changes while preserving recognition. |
| Stability level | Fixed core, variable presentation | Controls how much the graphic may change. |

## Icon State Variants

| State | Visual Goal | Rule |
|---|---|---|
| Unselected | Keep entry recognizable without stealing focus. | Use the base icon form and lower emphasis. |
| Selected | Confirm current navigation or active entry. | Add emphasis such as brand color, highlight, or selected shadow without changing semantics. |

| Pattern | Stable Part | Variable Part | Example |

## Scene Graphic Variants

| Pattern | Stable Part | Variable Part | Example |
|---|---|---|---|
| Festival benefit scene | Festival gift / benefit recognition | Composition density, decoration, warm color range | 聚宝盆 |
| Traditional festival scene | New year / ceremony recognition | Window lattice, blessing text, red-pink palette | 中式窗棂 |
| Delivery / home scene | Lantern and home-delivery emotion | Lantern shape, glow, background richness | 灯笼 |

## Category Graphic Variants

| Category Metaphor | Example Asset | Rule |
|---|---|---|
| Shopping / basket | 购物篮 | Keep object silhouette direct and scan-friendly. |
| Fashion / bag | 女包 | Preserve category recognition before decorative detail. |
| Beauty / skincare | 洗面奶 | Use product-like shape only when it remains legible at small sizes. |
| Digital | 数码 | Avoid over-complex inner detail. |
| Home / furniture | 沙发 | Preserve broad silhouette and avoid decorative clutter. |
