# Campaign System Materials Variants

## Variant Dimensions

| Dimension | Values | Purpose |
|---|---|---|
| Material type | Flat, color, realistic | Defines the visual expression family. |
| Expression dimension | 2D, 2.5D, 3D | Controls depth, light-shadow strength, and rendering complexity. |
| Campaign intensity | Daily, themed, immersive | Maps material strength to business scenario. |
| Source role | Inherited base material, marketing extension | Separates JD APP base material rules from campaign-specific expansion. |
| Asset treatment | Live style, static case image, exported source asset | Determines whether a material is implemented as CSS/style or preserved as an image. |

## Material Type Variants

| Type | Visual Feature | Application Feature | Suitable Scenario |
|---|---|---|---|
| Flat material | Relaxed, clear, universal. | Uses standardized graphics and spatial relationships to keep the image stable. | Daily campaign visuals and basic activity communication. |
| Color material | Bright, trend-driven, layered. | Uses color and shape combinations to strengthen attraction and visual impact. | Themed campaigns and atmosphere-enhanced visuals. |
| Realistic material | Textured, direct, immersive. | Strengthens realistic material and spatial expression to enhance atmosphere perception. | Immersive campaign scenes and high-emotion visual moments. |

## Inheritance Variants

| Inherited Material | Variation Rule | Review Point |
|---|---|---|
| Liquid Glass | Can be used as a translucent surface or layered highlight when the campaign visual needs depth. | Confirm contrast and text readability against the underlying image. |
| Frosted Glass | Can be used to create softened separation between information layers. | Confirm blur, opacity, and background complexity do not reduce recognition. |

## Application Patterns

| Pattern | Stable Part | Variable Part |
|---|---|---|
| Daily flat banner | Content structure, benefit recognition, clear hierarchy | Illustration style and simple color relationships |
| Themed color banner | Theme color, gradient direction, light effect rhythm | Decorative elements and image composition |
| Realistic atmosphere banner | Main object recognition, spatial depth, campaign emotion | Props, texture, light-shadow intensity, detail density |
