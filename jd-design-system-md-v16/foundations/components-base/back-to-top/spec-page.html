1组件定义

一句话：页面长列表滚动时浮现的单击回顶按钮，40×40 液态玻璃圆形悬浮件，承载一枚向上箭头，支持浅 / 暗双态。

页面长列表滚动时浮现的单击回顶圆形按钮 —— 40×40 液态玻璃（Liquid Glass）材质悬浮件，承载一枚向上箭头，点击将当前页面平滑滚回顶部。随系统外观切换浅 / 暗双态。

与 Toolbar / Tabbar 的边界

组件职责关系
返回顶部（本）长页滚动中的「回到顶部」快捷操作单一原子，悬浮于内容右下
底部工具栏 Toolbar业务页底部主操作承载栏常与之并存但不内嵌；层级更高、贴边浮起
Tabbar全局 tab 导航不重叠；返回顶部不参与导航，仅滚动控制
它不是导航、不是工具栏的一格，而是一枚独立悬浮控件：只做一件事 —— 回顶。

2行为准则

核心约定：只做回顶、不带文字；材质优先于实色；浅 / 暗各取独立材质与底衬；点击叠底衬给按压反馈。

  • 单一职责 —— 只做「回顶」，不堆叠次操作、不带文字标签 / 数字角标。
  • 触发回顶 —— 点击后当前页面平滑滚动回顶部（smooth scroll-to-top）。
  • 材质优先于实色 —— 底衬走「Liquid Glass - Small」材质，融入内容；不要用纯色圆按钮替代。
  • 深浅跟随系统 —— 浅 / 暗双态各取独立材质参数与底衬值，暗态不是浅态简单反相。
  • 按压即时反馈 —— 点击态靠叠加底衬（浅 plus_darker / 暗 plus_lighter），保持材质连续性。
⚠️ 出现 / 消失时机 Relay 稿未标注；建议口径：向下滚动超约 1 屏后浮现、回到顶部附近隐出，常驻内容右下角，待业务确认。

3设计属性

单一形态：40×40 液态玻璃正圆 + 20×20 上箭头。4 个展示态 = 浅 / 暗 × 默认 / 点击。暗色为独立态。

4应用场景

长列表 / 长页滚动场景的回顶快捷入口，浅 / 暗两套，悬浮内容右下、避让底部栏。

典型场景

场景说明
长列表回顶商品列表 / 信息流 / 搜索结果等长页，向下滚动超约 1 屏后浮现，点击平滑回顶。
浅 / 暗双态随系统外观切换；材质参数与图标色各取对应模式独立值。
与底部栏共存悬浮于内容右下，层级低于 Toolbar / Tabbar，避让底部安全区，不遮挡主操作。

Donts

❌纯色圆按钮替代

不要用纯色圆按钮替代 Liquid Glass 材质 —— 材质感是该组件的视觉契约。

❌加文字 / 角标

不要加文字标签 / 数字角标 / 次操作，保持单一职责。

❌暗态复用浅态

暗态不要直接复用浅态材质参数与图标色 —— 材质渐变 / 投影 / 底衬都换值。

❌点击态长亮

不要把「点击态」当持续选中态长亮 —— 它是按下瞬时反馈，松手即回默认态。

❌破坏正圆 / 命中区

不要改变 40×40 命中区或破坏 radius 20 正圆。
