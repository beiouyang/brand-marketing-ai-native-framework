# 布局 · V16.0

> V16 布局 = **Z 轴 4 层级** + **2 种页面框架**（卡片式 / 平铺式） + **双梯度间距**（导购型 / 平台型）。
>
> 数值规范见 [spacing.md](../tokens/spacing.md)；本文档讲**用法 / 决策依据**。

---

## 1. Z 轴层级 · 4 层模型 🚩 V16 新增显式定义

> V15 隐式分层，V16 把 Z 轴显式拆为 4 层，每层有明确交互行为和视觉规则。

### 层级（从下到上）

| 层级 | 用途 | 交互特征 | 视觉特征 |
|---|---|---|---|
| **背景层** | 氛围 / 大促换肤 | **无交互**，不可被位移 | 常态：中性背景色；大促：换肤烘托氛围；上滑后氛围适应 |
| **内容层** | 当前场景核心信息 + 核心任务 | 滑动、操作；被全局控制层覆盖 | 容器结构与背景层边界清晰；滑动符合物理运动逻辑 |
| **全局控制层** | 全局操作 + 导航功能 | 始终可见 / 浮于内容之上 | 顶导 / 底导 / 悬浮按钮 |
| **临时任务层** | 弹层 / 弹窗 / Toast | 进入时遮挡下层 | 用材质（Liquid / Frosted Glass）做层级感 — 见 [materials.md](./materials.md) |

### V16 场景拆解（规范页给的 3 个样本）

| 页面 | 背景层类型 |
|---|---|
| 首页 | 氛围背板 |
| 商详 | 主图背板 |
| 购物车 | 大促换肤背板 |

> 这 3 个场景在规范页都有 4 层拆解示意图，参考 Relay `6:3` 上的截图（搜列/商详/购物车/结算 各 3 版：原版 + before + after）。

### v15_status

- **v16_status: new** — Z 轴层级在 V15 没有显式规范，仅隐式存在于 visual/layout.md 的"屏幕分层"段落。V16 把 4 层模型作为基础视觉设定的第一条规则。

## 2. 框架布局 · 系统基准

> 基准设计尺寸：**375 × 812 DP**（iPhone 12/13/14 标准）

| 区域 | 高度 | 限制 |
|---|---|---|
| **iOS 顶部系统状态栏** | 44DP | 仅放沉浸式氛围，禁放操作 |
| **常规起始器**（标准 Tab Bar bottom safe area） | 34DP | 同上 |
| **悬浮底导起始器** 🆕 | **17DP** | V16 新增，悬浮底导浮在 home indicator 区域 |

---

## 3. 两种页面框架

> V16 把所有移动端页面分两类：**卡片式（导购型）** vs **平铺式（平台型）**。

### 3.1 卡片式布局 · 导购型

> **适用页面**：首页、搜推、我京、频道、商品列表、活动页
>
> **视觉策略**：圆角卡片收拢信息区块，聚焦视线进行有效导流

| 元素 | 间距 |
|---|---|
| 卡片距屏幕左右 | 8DP |
| 卡片之间纵向间距 | 8DP |
| **双列 Feeds 横向间距** | **7DP**（保证像素整除） |
| 顶导组件距屏幕左右 | 8DP |
| 卡片混排 容器纵向间距 | 8DP |
| 卡片混排 容器横向间距 | 7DP |
| 卡片横滑 容器纵向间距 | 8DP |
| 卡片横滑 容器横向间距 | 7DP |

#### 散落元素（未被卡片包裹）

- **距屏幕左右至少 16DP** —— 保证可操作性，避免曲面屏边缘干扰

### 3.2 平铺式布局 · 平台型

> **适用页面**：商详、购物车、结算、订列、订详、个人中心、设置页、后台
>
> **视觉策略**：直角拉通结构，最大化利用空间高效展示信息，减少层级嵌套

| 元素 | 间距 |
|---|---|
| 核心内容距屏幕左右 | **16DP** |
| 顶 / 底导组件 | 8DP |
| 纵向区块内容间距 | 8DP |

---

## 4. 元素布局 · 双梯度基量

> **基量标准**：标准元素以 **4 的倍率**逐步递增
>
> 间距与尺寸**双适用**（V15 仅 spacing 适用，V16 扩展到尺寸）

### 4.1 导购型梯度（紧凑）

> 信息密集，在有限空间内展示更多内容

间距：4 / 6 / 8 / 12（5 级语义见 [spacing.md](../tokens/spacing.md#22-导购型)）

### 4.2 平台型梯度（宽松）

> 操作明确，提升浏览舒适度与点击准确性

间距：4 / 6 / 8 / 12 / 16（5 级语义见 [spacing.md](../tokens/spacing.md#21-平台型)）
