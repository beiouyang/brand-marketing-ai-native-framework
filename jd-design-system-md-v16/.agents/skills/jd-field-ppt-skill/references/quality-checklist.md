# 质量检查清单

## P0 必须通过

- 每个 `.slide` 都有已登记的 `data-layout`。
- 页面没有明显文字溢出、遮挡、重叠。
- 标题是结论句，不是“背景/方案/总结”等空栏目。
- 红色 `#ff0f23` 只做强调，不能整 deck 泛红。
- 所有全局色值来自 `jd-color-tokens.md`，不得出现非 JD 基础色的主题 hex。
- 不出现紫蓝渐变、米色杂志风、暗蓝科技风、纯装饰光斑。
- 图片在槽位内，不能盖住页脚或正文。
- 页脚页码正确。

## P1 建议通过

- 每页只讲一个判断。
- KPI 有单位、口径或时间范围。
- 表格不超过 7 行，列名短。
- 页面左右/上下对齐稳定，留白一致。
- 至少一页体现真实京东业务对象。

## P2 体验优化

- 使用键盘左右翻页。
- ESC 或按钮可进入索引页。
- 打印/截图时背景稳定。
- 移动端可阅读封面和结尾页。
